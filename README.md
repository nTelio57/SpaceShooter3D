# Space Shooter 3D

PC Game made with Unity. Includes Client and Server projects.  
Game can be played singleplayer or multiplayer.  
Server uses custom made Web Socket.  

# Controls

### **Movement**  

**W** - Fly forward  
**S** - Fly backwards  
**A** - Fly left  
**D** - Fly right  
**Shift** - Speed boost  
**Ctrl** - Descent  
**Space** - Ascend  

### **Combat**  

**Left click** - Shoot rifle  
**Right click** - Shoot cannon  
**R** - Toggle laser
